Dispones de un fichero alumnos.txt con el siguiente formato:
nombre;nota
Ejemplo:
Juan;7.5
Ana;8.2
Luis;4.9
María;9.1
Juan;6.8
Desarrolla un programa en Java que:
1. Lea el fichero.
2. Almacene los datos en una colección.
3. Calcule:
o Nota media de cada alumno (pueden aparecer varias veces).
o Listado de alumnos aprobados.
o Listado de alumnos ordenados por nota media.
o Alumno con mejor nota media.
4. Guarde los resultados en un fichero resultado.txt.
Recomendaciones
• Usar BufferedReader y BufferedWriter (o Files).
• Usar colecciones
• Controlar posibles errores de lectura
